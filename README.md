# 🏠 AI-Powered House Price Prediction & Analytics

A Machine Learning web application that predicts house prices using the Ames Housing Dataset and provides interactive analytics through a Plotly dashboard.

## Features

- House Price Prediction
- Interactive Analytics Dashboard
- Price Distribution Analysis
- Correlation Heatmap
- Feature Importance Analysis
- Actual vs Predicted Visualization
- R² Score and RMSE Metrics

## Tech Stack

- Python
- Flask
- Scikit-Learn
- Pandas
- NumPy
- Plotly

## Dataset

Ames Housing Dataset (Kaggle)

## Run Locally

pip install -r requirements.txt

python app.py
